# Componentes Reutilizables

Esta sección almacena los componentes reutilizables de la aplicación.  
Si en algún caso se necesita un **pop-up** u otro componente que se use en varias secciones, debe colocarse aquí.  

Ejemplos de componentes reutilizables: 
- Modales (Pop-ups)  
- Tarjetas de información  
- Formularios reutilizables  
- Spinners o loaders
